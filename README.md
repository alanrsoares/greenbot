## Quick start

Use your favorite package manager to run greenbot

| package manager | command               |
| --------------- | --------------------- |
| npm             | `npx @greenbot`       |
| pnpm            | `pnpx  @greenbot/cli` |

With `yarn` you need to install it as a global dependency:

```bash
yarn global add @greenbot/cli
@greenbot/cli
```
